\doxysubsection{Detailed Description}
\subsubsection*{template$<$uint32 D, typename T$>$\newline
struct vec$<$ D, T $>$}

A vector struct of dimension, D and type T. 

A vector is a mathematical construct consisting of an ordered list of elements along with operations on it. 
\begin{DoxyTemplParams}{Template Parameters}
{\em D} & An integer representing the dimension of the vector. (ie. The number of elements in the ordered list) \\
\hline
{\em T} & A type, usually a numeric type. Each element in the vector will be of this type. T must be a type that overloads basic arithmetic operations. \\
\hline
\end{DoxyTemplParams}
