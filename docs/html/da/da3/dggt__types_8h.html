Detailed Description

Contains useful typedefs.

This file contains many useful type definitions mostly for brevity, (often removing the "_t" from the standard library's types. (eg. uin64_t becomes uint64).) but also for clarity (ie. float64 is more descriptive than double)

uint64_t bool64

A 64-bit integer type for use in boolean expressions.

This 64-bit integer type can be used as a boolean value or as 64 boolean values. Used in place of bool for memory alignment considerations.
