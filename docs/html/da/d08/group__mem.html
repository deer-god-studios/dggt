◆ scratch_alloc

template<u32 TABLESIZE = 2048, u32 LINEARSIZE = KB(10)>
using dggt::scratch_alloc = typedef allocator<alloc_t::FALLBACK,alloc_t::STORE_TABLE,TABLESIZE,alloc_t::LINEAR,LINEARSIZE>

A short way to write allocator<alloc_t::FALLBACK,alloc_t::STORE_TABLE,TABLESIZE,alloc_t::LINEAR,LINEARSIZE>.

This is a type of allocator where all memory comes off of the stack from a linear allocator and is recycled into a store table allocator.

Template Parameters
TABLESIZEThe number of memory recycling slots in the store table allocator. Defaults to 2048.
LINEARSIZEThe amount of memory off the stack to allocate from the linear allocator. Defaults to 10 KB.
